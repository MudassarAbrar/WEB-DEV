1. CSS is a style sheet language used for describing the presentation of a document written in HTML or XML.
  2. CSS is a cornerstone technology of the World Wide Web, alongside HTML and JavaScript.
  3. CSS is designed to enable the separation of presentation and content, including layout, colors, and fonts.
  4. CSS can control the layout of multiple web pages all at once.
  5. CSS can be added to HTML documents in 3 ways: inline, internal, and external.
    • this is a sub list
    • Inline CSS: CSS is applied directly to the HTML elements using the style attribute.
    • Internal CSS: CSS is defined within a (style) tag in the head section of the HTML document
    • External CSS: CSS is written in a separate file with a .css extension and linked to the HTML document using a tag.
  6. CSS selectors are used to select the elements you want to style.
